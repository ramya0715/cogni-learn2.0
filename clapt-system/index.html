How adaptive learning works here

  1. Observe. We capture accuracy, response time, retries and topic per answer.
  2. Score. A rule-based engine computes Cognitive Load (Low / Medium / High).
  3. Adapt. Difficulty and topic of the next question are adjusted instantly.
  4. Recommend. Weak topics get prioritised; strong topics get harder questions.
